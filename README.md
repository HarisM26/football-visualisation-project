# Premier League Data Visualisation

Interactive web app visualising 10 seasons of Premier League performance data using D3.js.

## Tech Stack
- **Frontend:** D3.js, JavaScript, HTML/CSS
- **Data Pipeline:** Node.js scripts for scraping and processing
- **Data Source:** Understat (xG, xA metrics)

## Features
- Interactive heatmaps, small multiples, and dumbbell plots
- Filter by team, season, and performance metrics
- Analyse team over/under-performance vs expected goals
